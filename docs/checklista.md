# Praktisk checklista inför primärvårdsarbete
## Första dagen: material och admin
- Stetoskop
- Namnskylt
- Arbetsskor
- Identifikationskort
- Inloggsdokument (tex RS-id inlogg: PUK och PIN-kod)
- Vid längre period: personliga föremål tex högtalare m.m.

## Vanlig dag
- Matlåda
- Almanacka

## Dagsschema
- 7:30 Byt om och ordna din arbetsyta
- 7:45 Morgonmöte och kaffe
- 7:50/efter möte: se agenda för dagen
- Patienter, se över agenda mellan patienter för att inte missa uppdateringar/ändringar
- 12:00: lunch och se över anteckningar som inte blivit av
- 13:00 patienter
- Innan hemgång: se över agenda!
